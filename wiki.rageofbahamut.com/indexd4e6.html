Welcome to Wiki -- Morgan (talk) 14:44, 12 July 2015 (EDT)&lt;editsection-brackets&gt;n">[edit]&lt;editsection-brackets&gt;

Hello Charlie2020,

and welcome to wiki. I've seen you've uploaded some pictures for Inspector-Priest Pyne. Please make sure the pictures you upload are using the right titel. For cards this would be "NAME OF THE CARD1.jpg", "NAME OF THE CARD2.jpg" and so on (without the "" of course ;) ). Wiki can only process the pictures you upload if they have the exact same name as the page they should belong to. Also: If you can't find pictures of a card online (Japanese Wiki for example) and you have to use pictures you have to take yourself, make sure to hide the ROB interface by presssing the "Menu"-Butten ingame for a short while. After that please use some sort of image editing software to cut out the picture from the rest of the frame before you upload the picture. I know it's a lot of work, but it saves space on the wiki server and other wiki users don't have to upload the pictures again. Keep up the good work and don't hesitate to ask questions if you've got problems with wiki. :) --Morgan (talk) 14:44, 12 July 2015 (EDT)

Editing Skills/Missing Skills -- Morgan (talk) 07:41, 25 November 2015 (EST)&lt;editsection-brackets&gt;n">[edit]&lt;editsection-brackets&gt;

Hey, thx again for your work on the wiki. I fixed Heroic Fighter. Just a tip for editing cards and skills gone missing:

I've seen you edit the top part of the page from

{{#vardefine: skill1234 | Heroic Charge}}

and split it up by writing it

{{#vardefine: skill123 | [Heroic Charge]}}
{{#vardefine: skill4 | [Heroic Blitz]}}

To prevent the skills from going missing you just have to adjust the lower part of the page. If you scroll down a bit you'll see the Cardset which wiki uses for the card's itself. If you change the parameter in the upper part, you must also change them in the cardset:

For example:


|name1 = {{PAGENAME}}
|rarity1 = {{#var:rarity123}}
|img1 = {{PAGENAME}}1.jpg
|trait1 = {{#var:trait1234}}
|atk1 = {{#var:atk_1}}
|def1 = {{#var:def_1}}
|maxatk1 = {{#var:max_atk1}}
|maxdef1 = {{#var:max_def1}}
|pwr1 = {{#var:pwr1234}}
|skill1 = {{#var:skill1234}}
|effect1 = {{#var:effect1234}}
|event_skill1 = {{#var:ev_sk123}}
|event_effect1 = {{#var:ev_ef123}}
|trait_skill1 = {{#var:trait_sk1234}}
|trait_effect1 = {{#var:trait_ef1234}}


Acording to the changes you made, the lines you have to change would be:

|skill1 = {{#var:skill1234}}
|effect1 = {{#var:effect1234}}

both variables must be changed from "1234" to "123" since you made a new line for skill4 and effect4. This change has to be made for the first 3 cards. The 4th card changes from "1234" to just "4" and would read:

|skill4 = {{#var:skill4}}
|effect4 = {{#var:effect4}}


This should prevent skills from going missing. I hope this is somehow understandable. If not, just ask again. (sry, English is not my first language) ;) --Morgan (talk) 07:41, 25 November 2015 (EST)

Thank you so much!
