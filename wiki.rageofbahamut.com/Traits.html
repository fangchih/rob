General[edit]

Traits were added to the game on May 29th, 2014. Each new card, as well as each existing card was assigned one of the following 5 Traits:

  • Agility
  • Charisma
  • Intelligence
  • Strength
  • Willpower


Trait Skills[edit]

  • When a card with trait skill is in the leader slot, the trait skill will always' activate. Card with trait skills on other slots will, however never activate.
  • The skill will boost all cards with the skill specific trait. This includes the trait skill card itself.
  • The activation of a trait skill is not included in the maximum activation of 3 skills. Therefor more than 3 skills can activate.
  • If a card has a trait skill and an event skill, both skills can activate in the same battle.
