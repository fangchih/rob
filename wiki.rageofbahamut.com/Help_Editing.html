Introduction[edit]

Thanks for wanting to contribute to the Rage of Bahamut Unofficial Wiki. We hope that the following information will answer any questions you had about contributing data.


Creating a Page[edit]

If you're reading this, you probably have already created a page.

The following information will assist you in how to enter card data using one of the 4 Templates available.

Brief Re-cap[edit]

Just use the searchbox (top right) and enter the base form card name (ensure proper spelling AND case sensitive *), event or mystery box (special card packs i.e. 1st Birthday). If it does not yet exists you will find a link like:

"Create the page "Daphne" on this wiki!"

In this case you can click on the card name to open a new and empty wiki-page. Add and fill out the template which is explained in the next chapter. Upload card images (you can get them from japanese wiki)

  • For more detailed information on adding a new Card page, Click here!
  • For information on how to add an Event page, click here!

Please read the directions when adding a new page. This will help to keep Wiki uniformed and consistent.


* Case sensitive - entering Sword Valkyrie is different from Sword valkyrie as is sword Valkyrie. Use the exact spelling and case as in game.'

Using Template 4[edit]

The following template performs calculations and so input of data is minimal.


Please use the templates from How to add a card as the are less clutter. The following are for informational purposes. Thanks. 


{{#expr:                         <!--DO NOT Change this. This allows for variables to be assigned to be used in the Cardset template.-->  
{{#vardefine: pwr1234   | ??}}   <!--Enter the Power of the card-->
{{#vardefine: rarity123 | ???}}  <!--For Forms 1, 2 & 3 - Options: Normal, High Normal, Rare, High Rare, S Rare, or SS  Rare-->
{{#vardefine: rarity4   | ???}}  <!--For 4th Form - Options: High Normal, Rare, High Rare, S Rare, SS  Rare, or Legend-->
{{#vardefine: skill1234 | ???}}  <!--Enter the card's skill-->
{{#vardefine: effect1234| ???}}  <!--Enter the card's effect-->

{{#vardefine: atk_1     | ???}}  <!--Enter the clean attack stats for the base card-->
{{#vardefine: def_1     | ???}}  <!--Enter the clean defense stats for the base card-->
{{#vardefine: atk_2 | {{ #expr: 1.2 * {{#var:atk_1}} }} }}    <!--DO NOT Change this. This performs the math for atk2-->
{{#vardefine: atk_3 | {{ #expr: 1.4 * {{#var:atk_1}} }} }}    <!--DO NOT Change this. This performs the math for atk3-->
{{#vardefine: atk_4 | {{ #expr: 1.8 * {{#var:atk_1}} }} }}    <!--DO NOT Change this. This performs the math for atk4-->
{{#vardefine: def_2 | {{ #expr: 1.2 * {{#var:def_1}} }} }}    <!--DO NOT Change this. This performs the math for def2-->
{{#vardefine: def_3 | {{ #expr: 1.4 * {{#var:def_1}} }} }}    <!--DO NOT Change this. This performs the math for def3-->
{{#vardefine: def_4 | {{ #expr: 1.8 * {{#var:def_1}} }} }} }} <!--DO NOT Change this. This performs the math for def4-->
