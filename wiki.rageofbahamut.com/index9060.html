Rupies/Stamina
Rupies/Stamina is a (very) rough calculation of how many rupies you would get per point of stamina spent on a quest, on average, if you sold the cards acquired.

(
 ((Man price + God price + Demon price + Rupie chest) / [4 if no treasure, 5 if treasure]) + 
 (Average rupies per attack * 3)
) / (Quest stamina cost * 3)

This formula does not take into account the relative rarity of silver chests/treasures, although it can be easily modified to do so if the probabilities become available.

All rupie values are automatically taken from their respective card pages. If any value is missing, please update the relevant card page directly.

Updated values for rupie chest rewards would be much appreciated - the formula has been adjusted to take them into account if they are available
