=Re: Re: Re: Re: Re: Missing Evolution Tables -- Vuice (talk) 00:41, 25 October 2013 (EDT)=&lt;editsection-brackets&gt;n">[edit]&lt;editsection-brackets&gt;
The 3 main pages that I modified for the evolution tables to work.

http://wiki.rageofbahamut.com/Template:StdEvolution

http://wiki.rageofbahamut.com/Template:StatType

http://wiki.rageofbahamut.com/Template:Multiplier

This was totally an educated guess on my part about how the multiplier page worked. Of course, you see the standard (STD) set of numbers, which applies for both ATK and DEF for a standard card. When I was looking into it, I was looking at how Beowulf/Dark Angel Olivia/Diana evo tables were being calculated, there were 4 sets of numbers ATK1/ATK2 and DEF1/DEF2 (which I cleaned up and renamed as the HERO set). The first set of numbers for ATK/DEF come from the multipliers of each base unenhanced in every evo that you find in the archives (e.g. evo1atk/evo1atk, evo2atk/evo1atk, evo3atk/evo1atk, evo4atk/evo1atk, etc.). Initially, I thought only 1 set of numbers was enough to get the correct evolution numbers in the table, so I got rid of the second set of ATK2/DEF2 numbers. Doing that ended up messing up the numbers by a much larger percentage than they were before. So I realized that the 2nd set of numbers are used to average the numbers out to their actual values in the game. I still don't fully understand how this is all calculated, but I didn't bother trying to after I figured this out. The one thing you do need to know is the actual values of a 4-7, 6-11, or 8-15 evolution of these cards in the game so that you can make the adjustments accordingly.

Making ATK1 set of numbers equal to the ATK2 set of numbers is the same as having one set of ATK numbers in the calculator. If the evolution table is off my a lot (e.g. anything greater than 1 point is considered off), then you need to modify the 2nd set of numbers to either be higher or lower to match the actual numbers of the cards you know of (e.g. the numbers of the 4-7, 6-11, or 8-15 that you know are correct in the game). These adjustments are usually off by less than 1%, so you just play around with the numbers until you get close enough.

I was able to make the numbers perfect for the Titans, but not the Brutes, because I don't know the actually numbers for a 8-15. I hope you're able to figure it out, because other than this, I have no idea how it's all calculated.
