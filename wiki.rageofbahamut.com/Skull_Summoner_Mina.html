Multiple Evolutions

Note: That the base form (Skull Summoner Mina) is stated as Form 0 and the max basic form level (Skull Summoner Mina+4) is called Form 4.

When multiple possibilities occur, the first # is the final form # followed by the combination done, See Final Form 2 for an example (21 = Form 1 is the last card used, 20 = Form 0 was used instead).

If this card is a higher form from a previous evolution., please remember to add 10% from the previous card to make the value of this card more accurate.

To do this, edit the source and where it says "MultiformEvolutionTable" and add the stats from the previous form so it is "MultiformEvolutionTable | Prev form ATK | Prev form DEF" . For example, adding the Schoolgirl Mina max stat (form 46M), it would be "MultiformEvolutionTable | 2222 | 1297".

If you do this, make sure you do 'SHOW PREVIEW' so the information doesn't vary for others :D - Ghostanime

Overview of some MAX Form Evolution

To understand what these form means, please see the full MAX evolution table below.

In the brackets, the value entered is the form used to evolve at that point. The value disregards the evolution to Skull Summoner Mina+

So, a (0102) means that the card was evolved in the following (bold = Form name): 

0 = 2 Skull Summoner Mina = Skull Summoner Mina+  (Form 10)
1  = Skull Summoner Mina+  +  Skull Summoner Mina+ <-- 1 = Skull Summoner Mina++ (Form 21)
0  = Skull Summoner Mina++ + Skull Summoner Mina <-- 0 = Skull Summoner Mina+3 (Form 310)
2  = Skull Summoner Mina+3 + Skull Summoner Mina++ <-- 2 = Skull Summoner Mina+4 (Form 433)

10 + 1 (10) = 21 + 0 = 310 + 2 (21) = 433
